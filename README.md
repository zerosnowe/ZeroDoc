# ZeroDoc

## <img title="" src="https://github.com/zerosnowe/ZeroDoc/blob/main/repo-image/Vector.png?raw=true" alt="" data-align="center" style="zoom:50%;">                                                                         ZeroDoc文档系统

### **⭐特点介绍**

1.🚀无需任何后端经验，纯HTML开发，小白都轻松上手

2.🐱依托GitHubAPI，只需配置好 GitHub Token 即可使用

3.🖊基于高性能文本引擎Markdown构建，扩展性超高

4.🍟UI基于MDUI V2组件库打造，更流畅，更好看



### **🚀快速部署**

##### 详细文档：咕咕咕🕊

1.Linux部署：使用1Panel+Openresty/Niginx部署（推荐）

2.Windows部署：SimpleWebServer部署/IIS部署

3.Cloudflare/Vercel托管部署（推荐）

4.☁雨云虚拟主机部署（推荐）
